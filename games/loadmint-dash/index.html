A crappy little endless runner by yours truly, effectively a clone of the Chrome dinosaur game featuring my good friend trickmint's fursona. Made in one day as a sort of gift to welcome him into nekoweb.

CC0 public domain license in the sense that you're free to put this up on your website without asking if you really want to for some reason, but son't expect me to release the actual GameMaker source code as it is kind of a mess.

How to play: Use the UP Arrow to jump over the broken images. That's it.
